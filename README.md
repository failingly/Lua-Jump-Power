# Lua-Jump-Power

## Loading the script
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/Stratxgy/Lua-Speed/refs/heads/main/speed.lua"))()
```



## Settings
```lua
getgenv().jump = {
    enabled = false,
    power = 50, 
    control = false,
    friction = 2.0,    
    keybind = Enum.KeyCode.KeypadDivide 
}
```
